# Blinkit Data Analysis Dashboard

## Project Overview

This project visualizes a comprehensive Power BI dashboard for analyzing Blinkit's sales performance, customer satisfaction, and inventory distribution. The dashboard provides key insights and opportunities for optimization using various KPIs and visualizations.

## Key Performance Indicators (KPIs)

The dashboard focuses on the following primary KPIs:

1. **Total Sales**: Overall revenue generated from all items sold ($1.20M)
2. **Average Sales**: Average revenue per sale ($141)
3. **Average Rating**: Average customer rating for items sold (3.9 out of 5)
4. **Number of Items**: Total count of distinct items sold (8523)

## Features

- **Filter Panel**: Allows users to filter data by outlet location type, outlet size, and item type
- **Outlet Size and Location Analysis**: Provides insights on sales performance by outlet size and location tier
- **Item Type Distribution**: Shows sales distribution across various product categories
- **Outlet Type Comparison**: Compares different outlet types based on sales, number of items, average sales, ratings, and item visibility
- **Outlet Establishment Trend**: Visualizes the growth of outlet establishments from 2012 to 2022
- **Fat Content Analysis**: Breaks down sales by low fat and regular fat products
